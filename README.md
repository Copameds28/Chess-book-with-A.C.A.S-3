## How to use:
1. Click "acas_user.js", then right-click the Raw button, and copy the URL link address
2. Install Tampermonkey or Violentmonkey extension
3. Go to the dashboard and click the + symbol, then click "Install from URL" and paste the URL you copied
4. Once you are on chess.com, click the extension and configure the features
5. Now enjoy :>
